3. 设置 JDK

Set the JDK version

假设你已经打开了一个项目

  1. 菜单栏依次选中 “File > Project Structure”
  2. 在 SDK location 页面的 JDK location 中选择你 JDK 所在的文件夹
  3. 选中 OK 保存

4. 设置桌面图标

此处有两种方式可以设置桌面图标：

  1. AS 内设置
    1. 运行 AS
    2. 选中 “Tools > Create Desktop Entry ”
  2. 自行编写 desktop 条目
